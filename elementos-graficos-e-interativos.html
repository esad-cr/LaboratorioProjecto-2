HTML5 - Elemntos Gráficos e Interativos

Canvas

O elemento <canvas> permite desenhar gráficos, animações, jogos e outros elementos gráficos diretamente no navegador, utilizando JavaScript.

Estrutura Básica

O navegador não suporta o elemento canvas (só é visível se o navegador não suportar o elemento canvas).
